# Current Plan Highlight Plan

## Goal

현재 시각이 포함된 계획을 원형 플래너에서 가장 먼저 인지되도록 강화한다.

## Scope

- 현재 계획 부채꼴 강조 강화
- 현재 계획 라벨 가독성 강화
- 현재 시간 포인터와 현재 계획의 연결감 강화
- 현재 계획 리스트 항목 강조 톤 정리
- 시계 숫자 라벨 단순화 및 가독성 강화
- 0시/12시 라벨 보조 아이콘 배치 조정

## Checklist

- [x] 현재 계획 섹터만 별도 시각 계층을 갖도록 SVG 렌더링 분기 정리
- [x] 비현재 섹터 대비를 한 단계 낮춰 현재 섹터가 먼저 보이게 조정
- [x] 현재 섹터에 추가 외곽선 또는 halo 성격 강조 추가
- [x] 현재 섹터 반지름 또는 외곽 경계를 소폭 확장해 선택된 조각처럼 보이게 조정
- [x] 현재 라벨 stroke/contrast를 보강해 밝은 색상에서도 읽히게 조정
- [x] 현재 시간 포인터 길이 또는 두께를 미세 조정해 현재 섹터와 연결감 강화
- [x] 리스트의 현재 계획 강조 톤을 원형 플래너와 같은 방향으로 조정
- [x] 시계 숫자를 `0, 3, 6, 9, 12, 15, 18, 21`만 보이도록 단순화
- [x] 숫자 라벨 색과 두께를 높여 가독성 강화
- [x] 0시/12시 라벨 옆 장식 아이콘 크기와 간격 조정
- [x] 현재 계획이 없을 때 과도한 강조가 남지 않는지 수동 확인
- [x] 모바일 폭에서 강조가 깨지지 않는지 수동 확인

## Notes

- 제품 목표는 사용자가 포인터를 읽기 전에 현재 계획 부채꼴을 먼저 식별하게 만드는 것이다.
- 강조는 단순 장식보다 현재 계획 인지 속도 개선에 집중한다.
- 현재 구현 브랜치는 `feature/current-plan-highlight`다.
- 수동 확인:
- `localhost:3000`에서 현재 계획 종료 시간을 임시로 `15:00`으로 낮춰 "계획 없음" 상태를 확인했다.
- `localhost:3001` DevTools responsive `400px` 폭에서 원형 플래너 강조, 숫자 라벨, 해/달 아이콘 배치가 유지되는 것을 확인했다.
