# Bootstrap Harness Plan

## Goal

빈 프로젝트를 에이전트 우선 개발이 가능한 초기 리포지터리로 전환한다.

## Scope

- git 리포지터리 초기화
- 핵심 문서 구조 생성
- 짧은 `AGENTS.md` 작성
- 목표 아키텍처 초안 작성
- 기획서 기반 제품 명세 1차 반영
- 최소 문서 검증 스크립트 추가

## Assumptions

- 제품명은 `오늘 다 했니?`다.
- 현재 리포지터리에는 구현 코드가 없다.
- 현재 확인한 기획 범위는 텍스트 파일에 적힌 초기 요약본이다.

## Steps

1. 리포지터리 초기화
2. 진입 문서와 docs 시스템 생성
3. 제품/품질/신뢰성/보안 기준 초안 생성
4. 문서 구조 검증 스크립트 추가
5. 다음 구현 단계 정의

## Risks

- 제품 요구사항이 실제 기획과 다를 수 있다.
- 구현 스택 선택을 성급히 확정하면 재작업이 생길 수 있다.

## Validation

- 필수 문서가 존재해야 한다.
- `AGENTS.md`는 짧은 맵으로 유지되어야 한다.
- 실행 계획과 제품 명세 문서가 연결되어야 한다.
- 제품 명세는 원본 기획서와 충돌하지 않아야 한다.
