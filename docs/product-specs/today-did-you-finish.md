## Product Hypothesis

`오늘 다 했니?`는 사용자의 생활계획표 실행을 돕는 습관 형성 지원 서비스다.

## Product Goal

최종 목표는 사용자가 하루 계획을 세우고, 지금 해야 할 계획을 즉시 이해하고, 실천과 완료 기록까지 이어지게 만드는 앱을 제공하는 것이다.

웹 MVP는 그 앱으로 전환하기 전에 핵심 사용자 흐름과 UX를 검증하는 단계로 사용한다.

## Interaction Direction

- 핵심 화면은 24시간을 원형으로 표현한 계획판이다.
- 현재 시각을 시침형 포인터로 표시해 지금 해야 할 계획을 직관적으로 보여준다.
- 일정은 분 단위로 자유롭게 생성 가능해야 한다.

## Candidate Core Journey

1. 사용자가 생활계획표 또는 오늘의 할 일을 등록한다.
2. 시스템이 적절한 시점에 알람 또는 리마인드를 제공한다.
3. 사용자가 각 항목의 완료 여부를 기록한다.
4. 사용자가 하루 또는 일정 기간의 실천 상태를 확인한다.
5. 사용자가 월간 동안 얼마나 지켰는지 보며 다음 실천 동기를 얻는다.

## Recovery Direction

- 종료 시각이 지난 미완료 일정은 `놓침` 상태로 분리해 다룬다.
- `놓침` 일정은 사용자가 선택적으로 `회고`하거나 같은 날 빈 시간에 다시 지정할 수 있어야 한다.
- 같은 일정의 재지정은 최대 3회까지만 허용한다.
- 원래 일정 길이를 채울 연속 빈 시간이 없으면 자동 축약이나 자동 분할 없이 다시 지정 실패로 처리한다.
- `완료`, `놓침`, `지금` 같은 표시 문구는 이후 사용자 설정으로 바꿀 수 있게 설계한다.
- 웹 MVP에서는 상태 버튼과 리마인드/놓침 액션 문구를 로컬 설정으로 바꾸고 유지할 수 있어야 한다.
- 회고가 비어 있는 `놓침` 일정과 곧 시작하는 다시 지정 일정은 다시 강조돼야 한다.
- 멀리 떨어진 후속 일정은 `다시 지정됨`, 임박한 후속 일정은 `다시 지정 곧 시작`으로 구분해 보여준다.

## Candidate User Value

- 계획을 실천으로 연결
- 완료 여부의 명확한 가시성
- 습관 형성을 위한 반복 사용 동기 제공
- 월간 완료 개수와 유지 흐름을 보며 장기 동기 유지
