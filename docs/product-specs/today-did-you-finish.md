## Product Hypothesis

`오늘 다 했니?`는 사용자의 생활계획표 실행을 돕는 습관 형성 지원 서비스다.

## Interaction Direction

- 핵심 화면은 24시간을 원형으로 표현한 계획판이다.
- 현재 시각을 시침형 포인터로 표시해 지금 해야 할 계획을 직관적으로 보여준다.
- 일정은 분 단위로 자유롭게 생성 가능해야 한다.

## Candidate Core Journey

1. 사용자가 생활계획표 또는 오늘의 할 일을 등록한다.
2. 시스템이 적절한 시점에 알람 또는 리마인드를 제공한다.
3. 사용자가 각 항목의 완료 여부를 기록한다.
4. 사용자가 하루 또는 일정 기간의 실천 상태를 확인한다.

## Candidate User Value

- 계획을 실천으로 연결
- 완료 여부의 명확한 가시성
- 습관 형성을 위한 반복 사용 동기 제공
