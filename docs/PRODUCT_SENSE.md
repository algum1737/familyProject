# Product Sense

## Working Product Hypothesis

`오늘 다 했니?`는 사용자가 생활계획표를 세우고, 알람을 통해 실천을 유도하고, 완료 여부를 기록하게 해주는 습관 형성 지원 제품이다.

## Confirmed From Planning Doc

- 기획 배경은 "계획만 하는 것이 아닌, 실천까지 할 수 있게 관리"다.
- 기획 목적은 습관을 만들기 위한 세밀한 계획과 실천 시스템 구축이다.
- 타깃 사용자는 일정을 계획하고 실천하는 사람들이다.
- 주요 기능은 생활계획표 등록, 알람, 완료 여부다.
- 플랫폼은 웹 MVP로 검증한 뒤 앱으로 전환하는 것이 우선이며, 웹은 최종 종착점이 아니다.

## Unknowns

- 로그인/사용자 계정 모델
- 가족 공유 여부
- 알람 전달 채널
- 오프라인 사용 여부
- 인증 필요 여부
- 동기화 요구사항
