Program overview

오늘 다 했니?

하루 계획을 세우는 데서 끝내지 않고, 지금 해야 할 일을 알아차리고, 완료하거나 다시 이어갈 수 있게 돕는 생활 계획 실행 앱입니다.

개인용 하루 계획 24시간 원형 플래너 로컬 리마인드 놓친 일정 회복 월간 동기부여

무엇을 해결하나

많은 계획 앱은 할 일을 적는 데 강하지만, 실제 시간에 맞춰 행동으로 이어지는 순간은 사용자가 직접 챙겨야 합니다. 오늘 다 했니?는 하루 계획을 시간대에 배치하고, 현재 시각 기준으로 무엇을 해야 하는지 바로 이해하게 만드는 데 집중합니다.

1

하루를 한눈에 보기

24시간을 원형으로 표현해 계획이 어느 시간대에 놓였는지 직관적으로 보여줍니다.

2

지금 할 일 알아차리기

현재 시각 포인터와 현재 계획 강조를 통해 사용자가 별도 계산 없이 지금 해야 할 일을 찾을 수 있습니다.

3

놓쳐도 다시 이어가기

미완료로 끝난 일정은 회고하거나 같은 날 빈 시간에 다시 지정할 수 있게 해 실행 흐름을 끊지 않습니다.

핵심 화면

현재 프로그램은 오늘의 실행을 중심으로 화면을 구성하고, 필요한 보조 흐름만 별도 화면으로 분리합니다.

오늘 화면

현재 계획 카드, 24시간 원형 플래너, 오늘 계획 목록, 완료 현황, 리마인드 배너가 모이는 기본 화면입니다.

계획 편집 화면

새 계획 추가, 기존 계획 수정, 놓친 일정 다시 지정을 같은 흐름에서 처리합니다. 제목, 시작 시간, 종료 시간, 색상을 입력합니다.

회고 화면

놓친 일정에 대해 왜 놓쳤는지 짧게 기록하고, 다음 행동을 이어갈 수 있게 돕습니다.

동기 화면

이번 달 완료 개수, 완료율, 놓친 일정 수, 연속 달성 흐름을 보여줘 다음 날에도 계속 열어볼 이유를 만듭니다.

사용 흐름

제품의 핵심 여정은 계획 등록, 현재 계획 인지, 완료 처리, 놓친 일정 회복, 월간 확인으로 이어집니다.

  1. 계획 등록 오늘 할 일을 제목과 시간으로 추가합니다.
  2. 시간 확인 원형 플래너에서 하루 배치와 현재 시각을 함께 봅니다.
  3. 리마인드 시작 5분 전과 종료 5분 전 알림으로 행동 타이밍을 놓치지 않게 합니다.
  4. 완료 처리 실행한 일정은 바로 완료로 바꾸고 오늘 완료 현황에 반영합니다.
  5. 회복 놓친 일정은 회고하거나 빈 시간에 다시 지정해 흐름을 이어갑니다.
  6. 동기 확인 월간 기록으로 내가 얼마나 지켰고 다시 이어갔는지 확인합니다.

주요 기능

첫 버전은 복잡한 협업이나 반복 일정이 아니라, 개인이 오늘 하루를 끝까지 실행하도록 돕는 기능에 집중합니다.

  • 분 단위 계획 생성 시작/종료 시간을 자유롭게 입력하고, 이미 등록된 시간과 겹치면 저장을 막습니다.
  • 현재 계획 강조 현재 시각이 포함된 계획 구간을 다른 일정과 구분해 보여줍니다.
  • 완료와 놓침 상태 완료하지 못한 일정은 종료 뒤 놓침 상태로 분리해 다시 볼 수 있게 합니다.
  • 회고와 다시 지정 놓친 일정은 회고 메모를 남기거나 같은 날 빈 시간으로 최대 3회까지 다시 지정할 수 있습니다.
  • 로컬 알림 Android 실기기에서 시작/종료 리마인드 예약, 알림 권한, 알림 채널, standalone 실행을 검증했습니다.
  • 월간 동기부여 완료 개수, 완료율, 놓침 수, 연속 달성 흐름으로 지속 사용 동기를 제공합니다.

현재 개발 상태

현재까지 웹 MVP와 Android 앱 흐름을 함께 구축하면서, 핵심 도메인 규칙과 화면 흐름을 검증해 왔습니다.

완료된 기반

오늘 계획 생성/수정/삭제, 완료 처리, 현재 계획 판정, 놓침 회복, 월간 요약, Android standalone 빌드와 실기기 알림 QA까지 진행했습니다.

남은 확장

출시 전에는 알림 문구 캡처 재확인, Play Console 제출 준비, 최종 회귀 QA, 실제 사용자 피드백 기반 UX 조정이 필요합니다.

제품 방향

오늘 다 했니?는 단순한 체크리스트보다 실행 순간에 더 가까운 앱을 목표로 합니다. 사용자가 계획을 적고 잊어버리는 대신, 지금 할 일을 알아차리고, 완료하고, 놓쳤더라도 다시 이어가게 만드는 것이 핵심입니다.
