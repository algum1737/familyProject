# Family Project

이 리포지터리는 하네스 엔지니어링 방식으로 프로그램을 구축하기 위한 초기 작업 공간이다.

현재 상태:
- Next.js + TypeScript 기반 웹 MVP와 검증 하네스가 있다.
- 제품 요구사항은 초기 기획서 텍스트 기준으로 1차 확인했고, 웹 MVP는 앱 전환 전 검증 단계로 본다.
- 제품명은 `오늘 다 했니?`다.

작업 원칙:
- 중요한 맥락은 리포지터리 안에 문서로 남긴다.
- `AGENTS.md`는 짧은 진입 맵으로 유지한다.
- 상세 규칙과 지식은 `docs/` 아래에 구조화한다.
- 규칙은 문서로만 두지 않고 가능한 한 검증 스크립트로 승격한다.
