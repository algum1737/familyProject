# 앱 개발 버전 작동법

이 문서는 `오늘 다 했니` Expo 앱을 개발 버전으로 직접 실행할 때 필요한 최소 절차만 정리한다.

기준 경로:

```bash
/Users/hun/workspace/familyProject/apps/expo
```

## 핵심 개념

현재 사용하는 앱은 `배포용 앱`이 아니라 `개발용 dev build`다.

그래서 보통은 아래 둘이 필요하다.

1. 기기에 설치된 Expo 개발 앱
2. JS 번들을 공급하는 Metro 서버

즉 `앱이 설치되어 있어도`, 개발 중에는 Metro 서버가 꺼져 있으면 앱이 정상 동작하지 않을 수 있다.

## 가장 단순한 사용법

앱만 볼 때는 아래 순서로 하면 된다.

### 1. Metro 서버 켜기

```bash
cd /Users/hun/workspace/familyProject/apps/expo
npm run dev
```

설명:

- 이 명령이 Expo Metro 서버를 띄운다.
- 보통 `localhost:8081`을 사용한다.
- 이 터미널은 앱을 보는 동안 계속 켜 둔다.

### 2. 시뮬레이터에서 앱 열기

- iOS 시뮬레이터에서 `오늘 다 했니` 앱을 직접 터치해서 연다.
- 이미 설치돼 있다면 보통 이 단계만으로 붙는다.

### 3. 끝나면 서버 끄기

```bash
Ctrl + C
```

## 처음 설치할 때

앱이 아직 시뮬레이터에 설치되지 않았으면 한 번 설치가 필요하다.

```bash
cd /Users/hun/workspace/familyProject/apps/expo
npx expo run:ios -d "iPhone 17 Pro"
```

설명:

- iOS dev build를 만든다.
- 시뮬레이터에 설치한다.
- 설치 후에는 앱 아이콘이 홈 화면에 생긴다.

주의:

- 이 명령은 `설치/재빌드` 용도다.
- 평소 앱만 확인할 때는 매번 할 필요 없다.

## 언제 다시 설치해야 하나

아래 경우에는 `npx expo run:ios ...`를 다시 하는 편이 맞다.

- 앱 아이콘을 바꿨을 때
- iOS 네이티브 설정을 바꿨을 때
- Expo 플러그인이나 native dependency가 바뀌었을 때
- 기존 설치 앱이 꼬였을 때

## 평소에는 무엇만 하면 되나

대부분은 이것만 하면 된다.

```bash
cd /Users/hun/workspace/familyProject/apps/expo
npm run dev
```

그 다음 시뮬레이터에서 앱을 연다.

## 자주 보는 상황별 정리

### A. 앱이 이미 설치되어 있다

```bash
cd /Users/hun/workspace/familyProject/apps/expo
npm run dev
```

그 다음 앱 열기.

### B. 앱이 설치되어 있지 않다

```bash
cd /Users/hun/workspace/familyProject/apps/expo
npx expo run:ios -d "iPhone 17 Pro"
```

설치가 끝나면 이어서:

```bash
cd /Users/hun/workspace/familyProject/apps/expo
npm run dev
```

### C. 홈 화면 아이콘이나 native 변경을 반영하고 싶다

```bash
cd /Users/hun/workspace/familyProject/apps/expo
npx expo run:ios -d "iPhone 17 Pro" --no-build-cache
```

그 다음 필요하면:

```bash
cd /Users/hun/workspace/familyProject/apps/expo
npm run dev
```

## 안드로이드일 때

Android 기기나 에뮬레이터를 쓴다면 설치 명령만 다르다.

```bash
cd /Users/hun/workspace/familyProject/apps/expo
npx expo run:android
```

하지만 개발 서버는 동일하게 필요하다.

```bash
cd /Users/hun/workspace/familyProject/apps/expo
npm run dev
```

## 한 줄 요약

평소 앱만 볼 때:

```bash
cd /Users/hun/workspace/familyProject/apps/expo && npm run dev
```

앱이 없거나 native 변경이 있을 때만:

```bash
cd /Users/hun/workspace/familyProject/apps/expo && npx expo run:ios -d "iPhone 17 Pro"
```
